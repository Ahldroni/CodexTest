# CodexTest

Static World of Warcraft Mythic+ progression dashboard for Ahldroni's characters.

## Hosting

The site is built for GitHub Pages and uses only static files:

- `index.html`
- `styles.css`
- `script.js`
- JSON snapshots in `data/`

Live URL:

```txt
https://ahldroni.github.io/CodexTest/
```

## Data

The frontend fetches `data/seasons.json`, then fetches the selected season file by relative path. No embedded `window.MPLUS_*` data or backend server is used.

Data is regenerated by GitHub Actions through:

```bash
node scripts/update-raiderio-data.mjs
```

The updater includes only characters with real Mythic+ score or runs for each specific season.

## Local Preview

Use any static file server:

```bash
python -m http.server 8000
```

Then open:

```txt
http://localhost:8000
```
